Flappy Bird Clone

A modern iOS implementation of the classic game

About the Game

A modern implementation of the classic Flappy Bird game using SwiftUI and SpriteKit. Experience the nostalgic gameplay with improved graphics and smooth animations.

Classic Gameplay

Enjoy the familiar and addictive gameplay mechanics that made Flappy Bird a worldwide phenomenon.

Modern Graphics

Beautiful visuals and smooth animations powered by SwiftUI and SpriteKit.

Score System

Track your progress with an intuitive scoring system and compete for high scores.

How to Play

  • Tap the screen to make the bird jump
  • Avoid hitting the pipes
  • Collect coins to increase your score
  • Try to get the highest score possible!

Get Started

To run the game on your iOS device or simulator:

  1. Clone the repository
  2. Open the project in Xcode
  3. Select your target device
  4. Build and run the project
